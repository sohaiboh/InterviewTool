Interviewtool Installation

WICHTIG!!!
Bitte lesen Sie erst nachfolgende Hinweise bevor Sie die Installation starten.

1. Voraussetzungen für die Installation von Interviewtool
Voraussetzung für die Installation von Interviewtool sind die Betriebssysteme Windows XP, Vista , Windows 7,Windows 8 oder Windows 10. Um TEO nutzen zu können, benötigen Sie das MSSQL Datenbank 852 oder neuere Version.
BITTE BEACHTEN! Haben Sie bereits die Vollversion von Datenbank installiert, sollten Sie die altere Datenbank nicht installieren oder nicht aktualisieren.

2. Empfehlungen
Wir empfehlen zum professionellen Arbeiten eine Bildschirmauflösung von 1366 x 768 Bildpunkten (oder höher) mit HighColor. Ebenso empfehlen wir eine tägliche Sicherung der Datenbank!

3. Einschränkungen von Interview ohne Lizenz

4. Testversion
 Sie dürfen die Software in der Testphase uneingeschränkt nutzen. Nach Ablauf der Testphase können Sie die Software durch Eingabe des Lizenzschlüssels zur Vollversion freischalten.  In der Testversion können sich maximal 1 Benutzer(Admin, Planer oder Teilnehmer) zeitgleich einloggen. Nach Eingabe eines Lizenzschlüssels können Sie mit Interviewtool und Ihren eingegebenen Daten uneingeschränkt weiterarbeiten.

5. Interviewtool Mail-Konfiguration 
Der Mail- Konfiguration speichert Ihre konfigurierte Mail-Server automatisch als SMTP Server Datei auf Ihrem Fileserver. konfigurierte Mail-Server können auf Wunsch direkt aus Planer-Seite geändert werden.
